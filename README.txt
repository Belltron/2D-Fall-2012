2D-Fall-2012
============
  
TAGD's first cross-platform 2D game for Fall 2012.  
  
Language:  
    Python has been chosen by the members of TAGD.  
        http://www.python.org  
        http://www.pygame.org  
  
Concept:
    This will be a 2D multiplayer tank game with a top-down perspective.
    Input from mouse will be used for aiming and attacks by left clicking.
    Movement forward and reverse will be keyboard keys W and S.
    Rotation of tank will be keyboard A and D.
    More to follow.
